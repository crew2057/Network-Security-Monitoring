## Objective
Performing threat hunting and detection techniques through the use of cyber threat intelligence and popular intrusion detection and data collection platforms. The goal is to identify signature and anomaly, and create rules for detection of anomalous activities on a network and host level.

## Skills Learned
- Conducted proactive threat hunting using CTI to identify IOCs and anomalies.  
- Developed and refined detection rules with Suricata and Wazuh to enhance threat detection.  
- Monitored and correlated security events with Security Onion and Kibana.  
- Analyzed traffic using Wireshark to identify suspicious patterns and intrusions.  
- Utilized Sysmon for endpoint visibility, investigating process activity and lateral movement.
